## Getting Started

### Prerequisites

- [Aptos CLI](https://aptos.dev/cli-tools/aptos-cli-tool/install-aptos-cli)
- [APM CLI](https://github.com/your-org/apm-cli)

### Building

```bash
aptos move compile
```

### Testing

```bash
aptos move test
```

### Publishing

```bash
apm publish
```
